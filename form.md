# 表单

## 1. action
```html
<form action="process.html">
</form>
```
action，表单提交给哪个页面进行处理

## 2. input text
```html
UserName: <input type="text" name="username">
```
- text文本类型
- 指定文本框名称username，用于提交数据

## 3. input password
```html
<input type="password" name="password">
```
- 指定文本框名称password，用于提交数据
## 4. input submit
```html
<input type="submit" id="">
```
- 显示为提交按钮
