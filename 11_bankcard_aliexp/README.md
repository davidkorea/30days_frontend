- text-decoration

  |值|描述|
  |-|-|
  |none|默认。定义标准的文本，没有样式|
  |underline|下划线，定义文本下的一条线。下划线 也是我们链接自带的|
  |overline|顶线，定义文本上的一条线|
  |line-through|删除线，定义~~穿过文本~~下的一条线|
